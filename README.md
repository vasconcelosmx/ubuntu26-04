# Ubuntu Server 26.04 LTS para Containers, Docker e Ambientes Cloud

## 📖 Sobre o Projeto

Este repositório foi criado com o objetivo de compartilhar gratuitamente procedimentos, documentação técnica e laboratórios práticos relacionados à instalação, configuração e administração do **Ubuntu Server 26.04 LTS**, utilizando máquinas virtuais no **Oracle VirtualBox**.

O conteúdo foi desenvolvido para estudantes, profissionais de infraestrutura, administradores de sistemas, analistas de suporte, DevOps e entusiastas que desejam aprender na prática como preparar um ambiente Linux moderno para utilização com containers, Docker, Kubernetes e serviços em nuvem.

Todo o material disponibilizado neste repositório é gratuito e foi elaborado com foco em aprendizado prático (*hands-on*), seguindo boas práticas utilizadas em ambientes corporativos.

---

## 🎯 Objetivos do Curso

Ao concluir os procedimentos apresentados neste repositório, você será capaz de:

* Realizar o download e instalação do Ubuntu Server 26.04 LTS;
* Criar e configurar máquinas virtuais utilizando Oracle VirtualBox;
* Entender os conceitos de ambientes On-Premises e Cloud Computing;
* Configurar servidores Linux para utilização em laboratórios e produção;
* Trabalhar com acesso remoto utilizando SSH;
* Preparar ambientes para Docker, Containers e Kubernetes;
* Conhecer as principais tecnologias utilizadas por Big Techs no Brasil e no mundo;
* Desenvolver habilidades práticas em administração de sistemas GNU/Linux.

---

## 📚 Conteúdo do Curso

### Módulo 01 - Download da ISO

* Download do Ubuntu Server 26.04 LTS;
* Verificação da arquitetura correta;
* Preparação da mídia de instalação.

### Módulo 02 - Criação da Máquina Virtual

* Instalação do Oracle VirtualBox;
* Criação da máquina virtual;
* Configuração de hardware virtual.

### Módulo 03 - Configuração da Máquina Virtual

* Ajustes de CPU e memória;
* Configuração de rede em modo Bridge;
* Configurações recomendadas para laboratórios.

### Módulo 04 - Instalação do Ubuntu Server

* Processo completo de instalação;
* Configuração de idioma e teclado;
* Configuração de armazenamento utilizando LVM;
* Criação de usuários e hostname.

### Módulo 05 - Primeiro Acesso ao Sistema

* Login no terminal;
* Validação dos serviços instalados;
* Primeiros comandos administrativos.

## 🖥️ Requisitos

### Hardware Recomendado

* Processador Dual-Core ou superior;
* 8 GB de RAM (mínimo recomendado);
* 50 GB de espaço livre em disco;
* Conexão com a Internet.

### Software

* Oracle VirtualBox 7.x ou superior;
* Ubuntu Server 26.04 LTS ISO;
* Sistema operacional Windows, Linux ou macOS.
